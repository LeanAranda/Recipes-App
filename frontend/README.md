# Recipes App - Frontend
Frontend en **Next.js + TypeScript** para la app de recetas.
Se conecta con el backend en NestJS para manejar usuarios, recetas y ratings.

## Stack
- [NextJS](https://nextjs.org/)                     - Framework frontend en Node.js
- [TypeScript](https://www.typescriptlang.org/)     - Tipado estático

## Levantar el proyecto
```bash
npm install
npm run dev
```
El frontend estará disponible en http://localhost:3000/
